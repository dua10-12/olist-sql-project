Этот проект представляет собой полный анализ данных маркетплейса Olist с использованием SQL.
Датасет включает более 100 тысяч заказов и детальную информацию о товарах, продавцах, клиентах и логистике.

# Структура провекта
```

olist-sql-project/
│
├── analytics/           # основные SQL-запросы
├── data/                # сырые данные (CSV)
├── diagrams/            # ERD, графики, визуализации
├── schema/              # описание структуры таблиц
└── README.md            # описание проекта
```
# Основные цели анализа

* Анализ динамики продаж (GMV, заказы, месячные тренды)
* Расчет AOV (average order value) и скользящего среднего чека
* Топ-категории, топ-товары, топ-продавцы
* Производительность продавцов
* Скорость доставки и проблемные регионы
* Анализ поведения клиентов (повторные заказы)
* Проверки качества данных
